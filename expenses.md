## Принцип деления — «очки»

Чтобы те, кто едет на 1–3 дня, не платили наравне с теми, кто на все 5, делим траты **по очкам пребывания в лагере**, а не поголовно.

**1 утро или 1 вечер в лагере = 1 очко.** Лагерные сутки:

- **Д1** — только ужин (заезд) → 1 вечер
- **Д2 / Д3 / Д4** — полные сутки → 1 утро + 1 вечер
- **Д5** — только завтрак (сборка/выезд) → 1 утро

Полная стоянка Д1–Д5 = **4 утра + 4 вечера = 8 очков**.

Две независимые шкалы:

- **Очки общака** — для всех 28 участников лагеря. Делятся **снаряжение, вода в магазинной фасовке, газ, дрова, санитария, мусор**, прочая инфраструктура.
- **Очки еды** — только для **22 чел общей кухни** (самоеды не платят). Делятся **продукты по [Меню](menu.md)**.

Кто во сколько входит — см. таблицы ниже. Подробности по срокам пребывания: [Участники](members.md), [Расписание](schedule.md).
